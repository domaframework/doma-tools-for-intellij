Checks whether the configuration for the custom function's defining class is properly set.

This inspection ensures that the required Doma configuration file and properties are correctly defined, and that the specified class implements org.seasar.doma.expr.ExpressionFunctions.

Conditions:

  • The Doma configuration file doma.compile.config exists directly under the resource directory.
  • The configuration file contains the property doma.expr.functions.
  • The class specified in the property implements org.seasar.doma.expr.ExpressionFunctions.

Example: 


// doma.compile.config
doma.expr.functions=com.example.CustomExpressionFunctions
